# expressblog

## Installing Project

1. git clone `https://github.com/alihandgrmnclr/expressblog.git`
1. cd expressblog
1. npm install
1. add your own `.env` folder and create an new variable that named `DB_URI` and add your own mongoDB link
1. nodemon app
1. go to `localhost:3000`

## Tech Stack

* NodeJS
* ExpressJS
* MongoDB
* MVC
